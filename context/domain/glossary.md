# Domain Glossary

## Card

Karta, której użytkownik szuka lub którą chce dodać do listy poszukiwań.

## Wanted Card

Karta brakująca lub poszukiwana przez użytkownika.

## Search Status

Aktualny etap poszukiwania karty.

Przykładowe statusy:

- searching
- contacted_seller
- waiting_for_reply
- ordered
- received
- problem
- cancelled

## Difficulty Score

Wynik określający trudność zdobycia karty.

## Priority

Priorytet działania użytkownika wobec danej karty.

## Seller Message

Wiadomość przygotowana dla sprzedawcy, opcjonalnie wygenerowana przez AI.
