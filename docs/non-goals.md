# Non-goals MVP

W MVP świadomie nie budujemy:

- integracji z Cardmarket API,
- automatycznego pobierania ofert,
- scrapowania marketplace'ów,
- płatności,
- publicznych profili użytkowników,
- systemu wymiany kart,
- aplikacji mobilnej,
- OCR zdjęć kart,
- rozbudowanych statystyk kolekcji,
- katalogu wszystkich istniejących kart.

Celem MVP jest zarządzanie poszukiwaniami brakujących kart, a nie budowa marketplace'u ani pełnego systemu kolekcjonerskiego.
